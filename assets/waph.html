  • Labs: Hands-on exercises in lectures
  • Lab 1: Foundations of the Web
    In web application programming, we delved into the functioning of the HTTP protocol on the web, analyzing it through Wireshark. The synchronous HTTP protocol comprises two main elements: the HTTP request and the HTTP response. In this process, the browser initiates communication with the server through an HTTP request, subsequently awaiting a response.
  • Lab 2: Front-end Web Development
    In this lab, we will use HTML to develop a basic webpage and JavaScript to add interactive features. You will also investigate jQuery, CSS, Ajax, and Web API integration to improve the look and feel of your webpage.
